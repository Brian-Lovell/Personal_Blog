Some HTML Code

Learning HTML is kind of like learning Legos. There are different blocks or elements that can be used to build a webpage. Each part of HTML is called an element. An element is the building blocks of the webpage that is organized by something called the Document Object Model or DOM for short.

The DOM works kind of like a tree. Each element is connected to the body or the tree trunk. The elements then branch out from the body like tree branches.

Elements of HTML

  • h1 is a header element!
  • p is a paragraph element!
  • ul is an unordered list!

With all the different elements assembled in an html document; the web browser is able to build a web page!
